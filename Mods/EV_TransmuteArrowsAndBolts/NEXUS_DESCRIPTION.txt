[size=6][b]Transmute Arrows and Bolts[/b][/size]

[size=5][b]Description[/b][/size]

Adds crafting recipes to convert arrows into crossbow bolts and vice versa at a [b]1:1 ratio[/b] with no material loss. Switch between bow and crossbow ammo without wasting resources.

[quote]
[color=#55aa55][size=4][b]Server-Side Friendly[/b][/size][/color]

[b]If you are running a dedicated server, this mod only needs to be installed on the server.[/b]
Players connecting to the server [b]do not need to download or install anything[/b] on their game clients — all changes are synced automatically when they join.
[/quote]

[size=5][b]Features[/b][/size]

[list]
[*][b]10 new recipes[/b] — 5 arrow-to-bolt conversions + 5 bolt-to-arrow conversions
[*][b]1:1 conversion[/b] — no material loss
[*][b]No XP gain[/b] — prevents exploit crafting for experience
[*][b]Tiered craft times[/b] — higher tier ammo takes slightly longer to convert
[*]Stone tier can be converted anywhere; Iron+ requires a Workbench
[/list]

[size=5][b]Conversion Recipes[/b][/size]

[size=4][b]Arrows to Bolts[/b][/size]
[list]
[*][b]Stone Arrow[/b] → Stone Bolt — 1s — Anywhere
[*][b]Iron Arrow[/b] → Iron Bolt — 1s — Workbench
[*][b]Steel AP Arrow[/b] → Steel AP Bolt — 2s — Workbench
[*][b]Flaming Arrow[/b] → Flaming Bolt — 3s — Workbench
[*][b]Exploding Arrow[/b] → Exploding Bolt — 4s — Workbench
[/list]

[size=4][b]Bolts to Arrows[/b][/size]
[list]
[*][b]Stone Bolt[/b] → Stone Arrow — 1s — Anywhere
[*][b]Iron Bolt[/b] → Iron Arrow — 1s — Workbench
[*][b]Steel AP Bolt[/b] → Steel AP Arrow — 2s — Workbench
[*][b]Flaming Bolt[/b] → Flaming Arrow — 3s — Workbench
[*][b]Exploding Bolt[/b] → Exploding Arrow — 4s — Workbench
[/list]

[size=5][b]Installation[/b][/size]

[list=1]
[*]Copy the [b]EV_TransmuteArrowsAndBolts[/b] folder into your game's Mods/ directory
[*]Restart the game or server
[/list]

[size=5][b]Compatibility[/b][/size]

[list]
[*]7 Days to Die 1.0 (Alpha 21+)
[*]Server-side mod — works without client installation
[*]Adds new recipes only — no vanilla recipe modifications, so conflicts are unlikely
[/list]
