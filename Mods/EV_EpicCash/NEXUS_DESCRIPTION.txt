[size=6][b]EV_EpicCash — Custom Server Currency[/b][/size]

[size=5][b]Description[/b][/size]

Adds [b]Epic Bucks[/b] ([font=Courier New]evCash[/font]) — a custom in-game currency designed for [b]dedicated server administrators[/b] who want to give their players a better experience. By offering item bundles through an in-game shop, admins can help newcomers and casual players catch up with veterans — letting everyone enjoy the server equally, regardless of how much time they spend in the game.

[quote]
[color=#55aa55][size=4][b]Server-Side Friendly[/b][/size][/color]

[b]If you are running a dedicated server, this mod only needs to be installed on the server.[/b]
Players connecting to the server [b]do not need to download or install anything[/b] on their game clients — all changes are synced automatically when they join.
[/quote]

[quote]
[color=#cc9900][size=4][b]Disclaimer[/b][/size][/color]

The author [b]does not endorse pay-to-win mechanics[/b]. This mod was created solely to help dedicated server owners sustain their communities and provide a fair, enjoyable gameplay experience for all players. If you choose to tie Epic Bucks to real-money donations, please [b]avoid aggressive, intrusive, or no-alternative monetization policies[/b]. The best servers are those where supporting the server is voluntary and never gives a decisive competitive advantage.
[/quote]

[size=5][b]How It Works[/b][/size]

Epic Bucks is a neon yellow in-game currency designed as a bridge between a server's external donation/points shop and the in-game economy. The flow is simple:

[list=1]
[*]A player purchases points or donates on the server's website.
[*]An admin (or automated system) issues Epic Bucks to the player via a console command.
[*]The player opens their crafting inventory in-game and exchanges Epic Bucks for the item bundles they want — no further admin involvement needed.
[/list]

This mod ships [b]92 ready-made bundles[/b] covering vehicles, weapons, ammo, armor, tools, skill magazines, skill books, workstations, food, and drinks. A built-in guide also explains how to add your own custom bundles for any combination of items at any price.

[size=5][b]Features[/b][/size]

[list]
[*][b]Custom Currency Item[/b] — [font=Courier New]evCash[/font] (Epic Bucks) based on the Old Cash model
[*][b]Neon Yellow Icon[/b] — instantly recognizable bright #FFF200 tint
[*][b]Large Stack Size[/b] — stacks up to 50,000 for convenient storage
[*][b]Cannot Be Sold[/b] — not sellable to traders, keeping it as a server-only economy
[*][b]Cannot Be Used as Fuel[/b] — prevents accidental burning
[*][b]Creative Menu Access[/b] — available in creative mode for easy admin distribution
[*][b]92 Ready-Made Bundles[/b] — vehicles, weapons, ammo, armor, tools, magazines, books, workstations, food, and drinks
[*][b]Color-Coded Categories[/b] — each bundle type has a unique neon icon tint for instant recognition
[*][b]Extensible[/b] — add unlimited custom bundles with any items and prices
[/list]

[size=5][b]Admin Commands[/b][/size]

Give Epic Bucks to a player:

[code]giveself evCash 100
give <player_name> evCash 100[/code]
