# Always Open Traders

## Description

Removes opening and closing time restrictions from all traders, making them available **24/7**. No more waiting around for the shop to open.

## Features

- All 5 traders are open around the clock
- Removes `open_time` and `close_time` attributes entirely
- No more locked doors at night — trade whenever you want

## Affected Traders

| Trader | ID |
|---|---|
| Trader Joel | 1 |
| Trader Rekt | 2 |
| Trader Hugh | 6 |
| Trader Jen | 7 |
| Trader Bob | 8 |

## Installation

1. Copy the `EV_ExtendedTraderHours` folder into your game's `Mods/` directory
2. Restart the game or server

## Compatibility

- 7 Days to Die 1.0 (Alpha 21+)
- Server-side mod — works without client installation
- May conflict with mods that modify trader `open_time`/`close_time` attributes
