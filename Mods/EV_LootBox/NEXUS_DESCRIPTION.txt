[size=6][b]Loot Box[/b][/size]

[size=5][b]Description[/b][/size]

Adds a [b]Loot Box[/b] — a special bundle item that opens a loot selection window with three named [b]reward categories[/b]. Each box rolls exactly one category: [b]Simple[/b], [b]Good[/b], or [b]Valuable[/b].

[quote]
[color=#55aa55][size=4][b]Server-Side Friendly[/b][/size][/color]

[b]If you are running a dedicated server, this mod only needs to be installed on the server.[/b]
Players connecting to the server [b]do not need to download or install anything[/b] on their game clients — all changes are synced automatically when they join.
[/quote]

[size=5][b]Features[/b][/size]

[list]
[*][b]Category-based probability system[/b] — each box rolls one reward category with 75% Simple, 20% Good, and 5% Valuable odds
[*][b]Matched weapon bonus[/b] — the selected category also has a 30% chance to add one weapon from the same reward category
[*][b]Loot Box[/b] ([font=Courier New]evLootBox[/font]) — single reward box with three named reward categories
[*][b]Loot window UI[/b] — uses the vanilla OpenLootBundle mechanic; opens a container window just like quest reward bundles
[*][b]No perks or loot abundance influence[/b] — rewards are fixed regardless of player progression
[*][b]Server-friendly[/b] — fully controlled by the server; admins distribute boxes via console commands
[/list]

[size=5][b]How It Works[/b][/size]

When a player right-clicks a [b]Loot Box[/b], the reward is built from one selected category plus an optional matching weapon bonus:

[list]
[*][color=#55aa55][b]Simple (75% roll)[/b][/color] — canned food, basic drinks, basic medicine, and 9mm Ball ammo
[*][color=#ffcc00][b]Good (20% roll)[/b][/color] — Meat Stew, First Aid Kit, and 9mm HP ammo
[*][color=#ff4444][b]Valuable (5% roll)[/b][/color] — Repair Kits, Steroids, and Handguns Skill Magazines
[/list]

The reward-category selector is a [font=Courier New]count="1"[/font] lootgroup with weighted entries. Each use of the box picks exactly one category: [font=Courier New]Simple[/font] with weight [font=Courier New]75[/font], [font=Courier New]Good[/font] with weight [font=Courier New]20[/font], or [font=Courier New]Valuable[/font] with weight [font=Courier New]5[/font]. The base consumables in that category are guaranteed, and then the matching weapon group rolls at [font=Courier New]30%[/font]. If that bonus roll succeeds, one weapon from the same reward category is added to the same loot window.

[size=5][b]Installation[/b][/size]

[list=1]
[*]Copy the [font=Courier New]EV_LootBox[/font] folder into your [font=Courier New]7 Days to Die/Mods/[/font] directory.
[*]Restart the game or server.
[*]Distribute boxes to players via the admin console:
[/list]

[code]giveself evLootBox 1
give <player_name> evLootBox 1[/code]
