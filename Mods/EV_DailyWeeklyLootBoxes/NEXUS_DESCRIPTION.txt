[size=6][b]Daily & Weekly Loot Boxes[/b][/size]

[size=5][b]Description[/b][/size]

Adds [b]Daily[/b] and [b]Weekly Loot Boxes[/b] — special bundle items that open a loot selection window with a [b]tiered reward system[/b]. Each box always grants a base reward tier and can independently add higher-tier bonus rewards on top.

[quote]
[color=#55aa55][size=4][b]Server-Side Friendly[/b][/size][/color]

[b]If you are running a dedicated server, this mod only needs to be installed on the server.[/b]
Players connecting to the server [b]do not need to download or install anything[/b] on their game clients — all changes are synced automatically when they join.
[/quote]

[size=5][b]Features[/b][/size]

[list]
[*][b]Tiered probability system[/b] — each box always grants a common reward and can independently add uncommon and rare bonus tiers
[*][b]Daily Loot Box[/b] ([font=Courier New]evDailyLootBundle[/font]) — lightweight daily reward with three quality tiers
[*][b]Loot window UI[/b] — uses the vanilla OpenLootBundle mechanic; opens a container window just like quest reward bundles
[*][b]No perks or loot abundance influence[/b] — rewards are fixed regardless of player progression
[*][b]Server-friendly[/b] — fully controlled by the server; admins distribute boxes via console commands
[/list]

[size=5][b]How It Works[/b][/size]

When a player right-clicks a [b]Daily Loot Box[/b], the reward is built in layers:

[list]
[*][color=#55aa55][b]Common (always)[/b][/color] — 2–3× Canned Sham, 3–5× Bandage, 50–100× 9mm Ball ammo
[*][color=#ffcc00][b]Uncommon (20% extra roll)[/b][/color] — 1–2× Meat Stew, 1–2× First Aid Kit, 50–100× 9mm HP ammo
[*][color=#ff4444][b]Rare (5% extra roll)[/b][/color] — 3–5× Repair Kit, 1–2× Steroids, 2–3× Handguns Skill Magazine
[/list]

The tier selector is a [font=Courier New]count="all"[/font] lootgroup. The common pool is always included, while the uncommon and rare pools use independent [font=Courier New]force_prob="true"[/font] rolls with [font=Courier New]prob="0.20"[/font] and [font=Courier New]prob="0.05"[/font]. If a bonus roll succeeds, [b]all items[/b] in that tier are added to the same loot window.

[size=5][b]Installation[/b][/size]

[list=1]
[*]Copy the [font=Courier New]EV_DailyWeeklyLootBoxes[/font] folder into your [font=Courier New]7 Days to Die/Mods/[/font] directory.
[*]Restart the game or server.
[*]Distribute boxes to players via the admin console:
[/list]

[code]giveself evDailyLootBundle 1
give <player_name> evDailyLootBundle 1[/code]
