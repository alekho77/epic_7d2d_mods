[size=6][b]ZZ_EV_ProjectZCash — EpicCash Recipes for Project Z[/b][/size]

[size=5][b]Description[/b][/size]

Adds [b]Epic Bucks[/b] crafting recipes for content from the [url=https://7dtdprojectz.com/]Project Z[/url] modpack. Players can spend Epic Bucks ([font=Courier New]evCash[/font]) to craft item bundles added by Project Z submods, turning the EpicCash economy into a gateway to Project Z's extended content. New bundle categories are added over time.

[quote]
[color=#55aa55][size=4][b]Server-Side Friendly[/b][/size][/color]

[b]If you are running a dedicated server, this mod only needs to be installed on the server.[/b]
Players connecting to the server [b]do not need to download or install anything[/b] on their game clients — all changes are synced automatically when they join.
[/quote]

[quote]
[color=#cc9900][size=4][b]Requires Two Base Mods[/b][/size][/color]

This modlet is an [b]add-on[/b] and does nothing on its own. It requires both base mods to be installed and loaded before it:

[list=1]
[*][url=https://github.com/alekho77/epic_7d2d_mods][b]EV_EpicCash[/b][/url] — provides the [font=Courier New]evCash[/font] currency used as the ingredient for all recipes.
[*][url=https://7dtdprojectz.com/][b]Project Z[/b][/url] — provides the items referenced by each recipe. Only the Project Z submods relevant to the currently added recipes are strictly required (see the [i]Included Recipes[/i] section below).
[/list]

The [font=Courier New]ZZ_[/font] folder prefix ensures the mod is loaded [b]after[/b] [font=Courier New]EV_*[/font] and [font=Courier New]Z_*[/font] mods (7D2D loads mods alphabetically by folder name), so all referenced items already exist when recipes are registered.
[/quote]

[size=5][b]Features[/b][/size]

[list]
[*][b]24 Project Z Recipes[/b] — 12 rare weapons, 2 rare tools, and 10 legendary unique weapons
[*][b]Always Unlocked[/b] — no schematic or perk level required
[*][b]Inventory Crafting[/b] — no workstation needed
[*][b]Tiered Pricing[/b] — rare gear in the 45–75 EB range, legendary gear priced very high (200–350 EB) to prevent instant end-game
[/list]

[size=5][b]Installation[/b][/size]

[list=1]
[*]Install [b]EV_EpicCash[/b] — copy [font=Courier New]EV_EpicCash[/font] into your [font=Courier New]7 Days to Die/Mods/[/font] directory.
[*]Install [b]Project Z[/b] — at minimum every submod referenced by the recipes listed above must be present in [font=Courier New]Mods/[/font].
[*]Copy the [b]ZZ_EV_ProjectZCash[/b] folder into the same [font=Courier New]Mods/[/font] directory.
[*]Restart the game or server.
[/list]

[size=5][b]Compatibility[/b][/size]

[list]
[*][b]Game Version:[/b] 7 Days to Die (latest stable)
[*][b]Type:[/b] XML modlet (recipes only — no items, no buffs, no Harmony)
[*][b]Multiplayer:[/b] Server-side only — clients do not need this mod installed.
[*][b]Load Order:[/b] The [font=Courier New]ZZ_[/font] prefix forces this mod to load after both [font=Courier New]EV_*[/font] and [font=Courier New]Z_*[/font] mods, which is required because recipes reference items defined in both.
[/list]
